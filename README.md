# XinnStore Pro Full Checked

Starter semi-production:
- Google login via NextAuth
- Supabase/Postgres schema
- Midtrans checkout + webhook route
- Digiflazz integration layer
- Profit engine
- Admin dashboard
- Order tracking

## Langkah awal
1. Copy `.env.example` ke `.env.local`
2. Jalankan `database/schema.sql`
3. Isi semua env
4. Set role admin di tabel `profiles`
5. `npm install`
6. `npm run dev`
